Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText property sets or returns the text content as plain text of the specified node, and all its descendants, The textContent property is used to get or set the text content of the specified node and its descendants.

Question-3: What is event delegation in the DOM?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

Question-4: What is event bubbling in the DOM?

Event bubbling can be defined as a phase of Event propagation in which if an event occurs on a particular element will propagate or bubble up to the ancestor.
